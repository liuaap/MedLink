# MedLink Page Structure

## Clinic Page（診所頁）

| 區 | 名稱 | 使用者問題 | 畫面內容 |
|------|------|------|------|
| CP-01 | 診所快覽 | 這間診所好不好？ | 返回、分享、收藏、診所照片輪播、診所名稱、評價、評論數、距離、診療項目標籤 |
| CP-02 | 診所介紹 | 這間診所在做什麼？ | 診所介紹標題、診所特色介紹文字、查看更多 |
| CP-03 | 推薦醫師 | 哪位適合我？ | 推薦醫師標題、查看全部（6）、推薦醫師卡、醫師照片、姓名、專長、評價、最快可約 |
| CP-04 | 診所資訊 | 我要怎麼去？ | 地址、電話、營業時間、停車資訊、地圖 |
| CP-05 | 診所操作 | 我要聯絡或前往診所？ | 導航前往、撥打電話 |
